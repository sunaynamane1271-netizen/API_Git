# API_Git

## Introduction

In this project i have developed Postman collection to test all CRUD operations using various features of postman like test/post-request scripts, PM library etc. 
In the Postman i had generated the script with the help of ai. Using pm library of javascript makes code easier to write. 
I had gone through the basic tools of postman for creation of file that's are collections, add folder, add request, selection of http methods (post, put, get), required URL, headers, body and script etc.
I hade used variables like Global , environment , collection and local variables.
